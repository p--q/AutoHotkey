Winキー＋Alt＋D  // 出力に用法がある
関数Eを呼び出し
if 先頭行が「--」から始まるときそれは外来処方オーダー
外来処方オーダーについて
関数Gを呼び出し
else if 先頭行が「処方日」から始まるときそれは入院処方オーダー
入院処方オーダーについて
関数Dを呼び出し
end if
関数Aを呼び出し
関数Fを呼び出し
改行を除くすべての空白を削除。
if 「^分[123]\S+」にマッチする文字列について
関数Bを呼び出し
else if 「^分\d\S+」にマッチする文字列について
関数Cを呼び出し
end if
関数Hを呼び出し
「整形完了(用法あり)」と表示
プレインテキストをクリップボードに戻す

関数H
「@@SPACE@@」を半角スペースに置換する
「\(\Sとして\)」を削除
「\(非持参\」を削除


関数G
「--」「<R」「処方箋」から始まる行と空行は削除。

関数E
選択範囲をコピー。選択範囲にテキストがなければコピーせずにクリップボードのプレインテキストを使用。
クリップボードに取得したプレインテキストについて
全角文字を半角に変換（カタカナも括弧も半角にする）


関数D
「処方日」から始まる行を区切りとしてその行は含めずに一つの処方として取得。
一つの処方について
行を上から順に処理する
「「\d+\S*[錠pg枚ﾄ]$」にマッチする行」＝トリガー行
トリガー行が出てきたら、それまでに連続していた
「スペースを含まない行」たちを全部結合して1行にする
その結合結果の後ろに トリガー行も続けて結合 する
次のトリガー行まで同じ処理を繰り返す
「処方日」から始まる行は破棄




関数A
「\d+\S*分$」にマッチする文字列を削除。
「\d+\S*[錠pg枚ﾄ]$」にマッチする文字列の前に「@@SPACE@@」を挿入。
「cap$」にマッチする文字列を「c」に置換。

関数F
「^\S+時」にマッチする文字列は上の行に結合する
「^分\d+\s\d」にマッチする文字列の\sを「@@SPACE@@」に置換する
「^外)/s」を「@@SPACE@@」に置換して上の行に結合
「^吸入用」を削除。




関数B
「毎(?=.)」と「食後」は削除、
「(?:と)?眠前」を「寝」に置換
「食前」を「前」に置換
「\[食間\]」は削除
「1日\d回」を削除
上の行と結合する

関数C
「(?:と)?眠前」を「寝」に置換
「\[食間\]」は削除
「1日\d回」を削除
上の行と結合する
