

Winキー＋Alt＋S  // 出力に用法がない
関数Eを呼び出し
if行頭に「商品名」があるときそれはDI情報
DI情報について
「商品名」とそれに続く空白を削除。
else
関数Dを呼び出し
「\d+\S*[錠pg枚ﾄ]$」にマッチする行以外を削除
「\d+\S*[錠pg枚ﾄ]$」にマッチする文字列の前に「@@SPACE@@」を挿入。
「cap$」にマッチする文字列を「c」に置換。
改行を除くすべての空白を削除。
「@@SPACE@@」を半角スペースに置換する
end if
「\(\Sとして\)」を削除
プレインテキストをクリップボードに戻す


Winキー＋Alt＋D  // 出力に用法がある
関数Eを呼び出し
関数Gを呼び出し
関数Aを呼び出し
関数Fを呼び出し
改行を除くすべての空白を削除。
if 「^分[123]\S+」にマッチする文字列について
関数Bを呼び出し
else if 「^分\d\S+」にマッチする文字列について
関数Cを呼び出し
end if
「@@SPACE@@」を半角スペースに置換する
「\(\Sとして\)」を削除
プレインテキストをクリップボードに戻す

関数G
if 先頭行が「--」から始まるときそれは外来処方オーダー
外来処方オーダーについて
「--」「<R」「処方箋」から始まる行と空行は削除。
else if 先頭行が「処方日」から始まるときそれは入院処方オーダー
入院処方オーダーについて
関数Dを呼び出し
「処方日」で始まる行を削除
end if


関数E
選択範囲をコピー。選択範囲にテキストがなければコピーせずにクリップボードのプレインテキストを使用。
クリップボードに取得したプレインテキストについて
全角文字を半角に変換


関数D
「処方日」から始まる行の次の行から次の「処方日」から始まる行または最終行までを一つの処方として取得。
一つの処方について
行を上から順に処理する
「行の中にスペースが1つ以上ある行」＝トリガー行
トリガー行が出てきたら、それまでに連続していた
「スペースを含まない行」たちを全部結合して1行にする
その結合結果の後ろに トリガー行も続けて結合 する
次のトリガー行まで同じ処理を繰り返す




関数A
「\d+\S*分$」にマッチする文字列を削除。
「\d+\S*[錠pg枚ﾄ]$」にマッチする文字列の前に「@@SPACE@@」を挿入。
「cap$」にマッチする文字列を「c」に置換。

関数F
「^\S+時」にマッチする文字列は上の行に結合する
「^分/d+\s\d」にマッチする文字列の\sを「@@SPACE@@」に置換する
「^外)/s」を「@@SPACE@@」に置換して上の行に結合
「^吸入用」を削除。




関数B
「毎(?=.)」と「食後」は削除、
「(?:と)?眠前」を「寝」に置換
「食前」を「前」に置換
「\[食間\]」は削除
「1日\d回」を削除
上の行と結合する

関数C
「(?:と)?眠前」を「寝」に置換
「\[食間\]」は削除
「1日\d回」を削除
上の行と結合する
